## Project Summary
I developed this Power BI dashboard to analyze sales performance, customer behavior, and product trends. The goal was to convert raw business data into meaningful insights that can support decision-making.

## What I Built
- Created data model and relationships between tables
- Developed DAX measures for key business KPIs
- Designed an interactive dashboard with slicers and filters
- Implemented drill-through functionality for customer-level analysis
- Built visualizations to track sales trends and segment performance

## Key Metrics
- Total Sales
- Total Orders
- Return Rate
- Total Units Sold

## Dashboard Insights
- Analyzed sales performance across customer segments
- Identified top-performing customers and products
- Tracked monthly sales trends
- Compared category-level performance
- Enabled detailed customer-level investigation through drill-through pages
